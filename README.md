# Game Of Life
John Horton Conway's Game of Life Simulation

### Requirements
> **Pygame** : pip install pygame

> **Numpy**  : pip install numpy
---
### COMMANDS
        "Esc"             : to close the window
        "Space"           : to Pause the simulation
        "Mouse Left Click": to draw on the grid
